# xz_spider

## 介绍  

一个简单的写真爬虫   
a simple spider which can get girl photo.  
这是一个使用Python编写的爬虫，目前仅在Python3.7上测试过。
代码可能较不规范。但我会尽量把它写规范的(超大声🤗  
  
## 📖使用  

**1.安装所需包**  

```shell script 
pip install -r requirements.txt
```

**2.运行getxz.py**  
```shell script 
python3 getxz.py
```  
**3.坐等，爬到的文件在当前目录的data目录中**
